# Castelino Capital

A multi-asset macro hedge-fund simulation built as a LangGraph DAG of LLM agents. Trades equities, fixed income, commodities, and FX through a hypothesis-first funnel, opposing Bull/Bear analysts, a constitutional governance layer, and a custom mark-to-market simulator.

**Status:** Design phase. See [`docs/plans/2026-05-03-castelino-capital-design.md`](docs/plans/2026-05-03-castelino-capital-design.md) for the full architecture.
